Computer Organisation Tutorials



Welcome to COT (Computer Organisation Tutorials), an online learning platform for the study of internal working, structuring and implementation of a computer system. Computer Organisation is concerned with the structure and behaviour of digital computers. The main objective of this learning platform is to help students and interested readers understand the overall basic computer hardware structure, including the peripheral devices. Computer Organisation is about the design of different hardware computer parts and how they come together to form a computer system.

To further understand the practical application of the theoretical concepts, the website includes a section on algorithm implementation using 8085 assembly language. Assembly languages are low-level languages used for direct communication with the components of the machine.

In the midst of a pandemic, it is important to stay updated with the current happenings, therefore the website also contains a section which provides real time data about the same.
